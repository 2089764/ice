# ice

PHP-Web开发框架

##基础功能开发

* ✓ 三种运行方式: web, service, daemon
	
* ✓ 四种请求数据结构: Request, Response, ClientEnv, ServerEnv
	
* ✓ 两个框架基础封装: Ice, App
	
* ✓ 一个资源管理机制: $app->proxy_resource
	
* ✓ 一个服务管理机制: $app->proxy_service
	
* ✓ 一组框架基础工具集: Logger, Config
	
##外围工具开发

* ✓ 一套基础工具库: Util
	
* ✓ 一套基于mysqli的数据库工具集: DB_Query
	
* ✓ 一套数据过滤工具集: $app->proxy_filter
	
##协议规范

* 一套分层规范
	
* 一套静态代码检查工具
	
* 一份编码规范文档
	
* 一份架构介绍文档
	
* 一套Example使用示例
